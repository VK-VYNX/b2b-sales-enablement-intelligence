# VYNX Workshop Agenda

Use this file when the user needs a live workshop design for a client team.

## Workshop Title

High-Value Deal Conversion Design Session

## Attendees

- commercial leader
- channel leader or distributor manager
- direct sales leader
- 2-3 regional sales managers
- applications engineering leader
- service or implementation leader
- RevOps or CRM owner
- optional: one high-performing distributor representative

## 90-Minute Agenda

### 0-10 min: Alignment

- objective: agree the problem and define scope
- exercise: rapid fact-pattern review
- output: problem statement and target deal type

### 10-25 min: Leakage Mapping

- exercise: map where high-value deals are dying across RFQ, visit, validation, proposal, and commercial review
- output: stage leakage view

### 25-45 min: Failure Pattern Diagnosis

- exercise: review 3 recent deals: won, lost, stalled
- output: root-cause themes across qualification, handoff, service confidence, and distributor capability

### 45-65 min: Operating Model Design

- exercise: define stage gates, owners, and escalation triggers
- output: draft stage-gate architecture and decision rules

### 65-80 min: Management System Design

- exercise: define weekly, biweekly, and monthly cadence plus dashboard KPIs
- output: review cadence and KPI pack

### 80-90 min: Commitment and Pilot Scope

- exercise: select pilot region, distributors, metrics, and timeline
- output: pilot decision and named owners

## Decisions Expected By The End

- which deal types enter the operating system
- which stages and gates will be used
- when applications engineering and service must be engaged
- which distributors will be piloted
- which KPIs leadership will inspect monthly
