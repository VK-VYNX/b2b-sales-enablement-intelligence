# Category: Sales Enablement Technology

Use this file for technology stack, tool evaluation, integration, and workflow support.

## Expected Knowledge From This Category

- enablement technology categories
- tool selection criteria
- implementation and adoption risks
- integration logic
- workflow support

## Distillation Priorities

Prioritize extraction of:

- decision criteria for choosing tools
- process prerequisites before tool adoption
- owner model and governance
- data and workflow integration considerations
- adoption and value measurement
- practical AI use cases that support real workflows
- criteria for separating useful automation from hype

## Common Misread

Technology should be framed as a support layer, not the primary answer to a behavioral or ownership problem.
