# Category: Sales Coaching

Use this file for manager enablement, inspection, feedback loops, and behavior reinforcement.

## Expected Knowledge From This Category

- coaching system design
- manager role in enablement
- observation and feedback
- reinforcement cadences
- coaching quality metrics

## Distillation Priorities

Prioritize extraction of:

- conditions under which coaching changes behavior
- manager responsibilities versus enablement responsibilities
- coaching process, not just coaching advice
- linkage between methodology, call review, and pipeline review

## Common Misread

Do not confuse occasional feedback with a coaching system.
