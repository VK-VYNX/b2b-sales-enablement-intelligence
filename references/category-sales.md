# Category: Sales

Use this file for adjacent sales topics that inform enablement decisions but are not enablement-specific.

## Expected Knowledge From This Category

- sales performance drivers
- pipeline health
- role outputs
- GTM context
- collaboration metrics

## Distillation Priorities

Prioritize extraction of:

- performance signals relevant to enablement
- role expectations that shape readiness and coaching
- collaboration logic between enablement and field teams

## Common Misread

Not every sales problem is an enablement problem, but many enablement decisions should be grounded in sales system realities.
