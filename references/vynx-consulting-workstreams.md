# VYNX Consulting Workstreams

Use this file when the user wants the methodology translated into consulting workstreams or implementation tracks.

## Pipeline Diagnostics

Purpose:
- isolate where strategic deals break down

Core activities:
- opportunity audits
- stage leakage analysis
- loss and stall review
- distributor comparison
- stakeholder interviews

Outputs:
- diagnostic heatmap
- failure-pattern summary
- KPI baseline

## Process and Stage-Gate Design

Purpose:
- define how high-value deals should move

Core activities:
- stage architecture
- exit criteria
- escalation logic
- proposal block rules
- ownership definition

Outputs:
- stage-gate scorecard
- decision-rule set
- operating model map

## Distributor Performance Management

Purpose:
- control channel variance and align support to capability

Core activities:
- partner segmentation
- tier scoring
- compliance logic
- support model design
- consequence definition

Outputs:
- distributor tiering model
- partner scorecards
- review cadence

## AE and Service Handoff Design

Purpose:

Core activities:

Outputs:
- technical-commercial handoff brief
- service engagement rules
- handoff acceptance standard
