# Manufacturing Use Cases

Use these examples to keep outputs grounded in industrial and engineering-led selling environments.

## Example Requests

- Diagnose why our distributor network generates quotes but does not convert high-value machinery deals.
- Build a 90-day enablement plan for an industrial automation company selling through direct reps and system integrators.
- Design onboarding for technical account managers selling engineered solutions into manufacturing plants.
- Create a coaching system for regional sales managers who lead field reps and applications engineers.
- Define KPIs for RFQ response quality, plant-visit effectiveness, and proposal-to-order conversion.
- Build a content governance model for technical decks, case studies, calculators, and specification sheets.
- Recommend a methodology rollout for a manufacturer with long cycles, multiple stakeholders, and distributor influence.

## Example Output Shapes

### Distributor performance diagnosis

1. Situation
2. Likely root causes
3. Distributor enablement interventions
4. Field-management changes
5. Metrics to track

### RFQ improvement plan

1. Current failure points
2. Required handoffs
3. Proposal quality standards
4. Review cadence
5. KPI framework

### Plant-visit enablement system

1. Required preparation
2. Discovery checklist
3. Stakeholder map
4. Post-visit follow-up sequence
5. Opportunity progression signals
