# VYNX Industrial Sales Enablement Intelligence

Reusable agent skill and portable knowledge pack from VYNX for Industrial Engineering and Manufacturing sales enablement strategy, diagnostics, operating models, measurement, coaching, training, content, methodology, and technology decisions.

## What this package contains

- `SKILL.md`: native skill instructions for Codex-style agents
- `agents/openai.yaml`: skill UI metadata
- `references/`: modular intelligence files for routing and reasoning
- `portable/core-intelligence.md`: agent-agnostic knowledge pack
- `exports/single-file-agent-pack.md`: single-file export for tools that prefer one document
- manufacturing-specific guidance, use cases, and output shapes

## Intended use

Use this package when an agent needs to:

- diagnose commercial execution problems in industrial and manufacturing businesses
- design or improve an enablement function for direct sales, distributors, channel partners, or technical sales teams
- define KPIs and measurement systems for long-cycle, multi-stakeholder deals
- recommend training, coaching, content, methodology, or technology interventions for engineering-led sales environments
- assess stakeholder alignment or maturity across sales, applications engineering, product, operations, service, and channel teams

## Domain focus

This package is built for environments such as:

- industrial automation
- machinery and equipment
- engineered products
- components and systems
- industrial software sold into manufacturing environments
- process manufacturing support solutions
- channel and distributor-led industrial sales models

## VYNX positioning

VYNX is positioned here as a B2B RevOps and revenue enablement operator for complex industrial growth systems. The package is designed to sound like VYNX: practical, commercially sharp, operationally grounded, and implementation-first.

## Design principles

- concise, implementation-ready guidance
- clear separation of symptoms, causes, interventions, and metrics
- strong emphasis on operating logic and sequencing
- portable across agent systems
- domain-specific examples for technical validation, RFQs, plant visits, pilots, distributor enablement, and multi-stakeholder deals

## Recommended sharing model

- Use the full folder for Codex-style skill systems
- Use `portable/core-intelligence.md` for agent workflows that prefer modular knowledge packs
- Use `exports/single-file-agent-pack.md` for systems that only accept one file

## Notes

This public package is intentionally clean-room and source-agnostic. It contains generalized frameworks, original synthesis, and VYNX-branded industrial adaptation only.
